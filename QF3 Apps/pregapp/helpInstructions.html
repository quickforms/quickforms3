Content Editing Tips

1.     Text.

a)     Text Format. The font and its size used in each tab and week should be compliant with each other. The default font recommended by Francine is Arial, size in number 2 of the HTML editor. Source text information should be made italics beginning with a short " - " (dash) in a separate line.

Caveat: please pay special attention to the apostrophe mark or other special mark in the text, once you submit the text, it may not be recognized as an "apostrophe" or a "dash" while all showed in a "question mark". You will need to manually change it to with a second submission.

 

b)     Pasting Text from other sources. While pasting text from other sources like Microsoft Word, it is recommended to paste as plain text (Shift+Ctrl+V at Chrome) in the HTML editor to avoid potential page-down problems. In the Mac platform, use "Shift+Command+V" to paste text in Chrome browser.

 

2.     Image.

Please follow these steps to upload images:

a)     Upload images to the server. Folder path:

C:\Users\Public\Documents\Tomcat7\webapps\pregapp\img\content

b)     In the HTML editor, click the last icon which says "show source". Use the following code to retrieve images from the server. The relative path is "img/content/" + file name as the path for pictures (replace space with %20). For example, if you try to use "perfect family shot web.jpg", the full path would be "img/content/perfect%20family%20shot%20web.jpg" with the <img> tag.

 

3.     Hyperlinks.

Please bear in mind that when copying and pasting hyperlinks, the app sometimes do not automatically paste the hyperlink. You need to use the hyper link function in the HTML editor to add the link onto the text you want to be hyper linked.
