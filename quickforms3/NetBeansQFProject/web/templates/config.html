Required config.js file

Create a config.js file in [tomcat dir]/webapps/[app name]/js/

Paste the following lines of code in the config.js file and replace the 'AppName' variable

define(function(){
var config = {};
config.app = 'AppName';
return config;
});

Full example here

If no authentication is required, add the following line before return config;
config.extraScripts = [];
