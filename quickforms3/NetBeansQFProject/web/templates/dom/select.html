Measure Selects:

In the event that a select element is required in an aggregation statement (sum, average, ect.), add the "measure" attribute in the select tag.
To capture this in the database, add a column "[lookupName]Measure" to the fact table. Eg. ageMeasure

Remember Selects:

Preserves selection between refreshes and page navigation. Default memory length set to one day. Check master config file to change this setting.
